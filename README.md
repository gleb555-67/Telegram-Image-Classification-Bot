# top_bot

## Привет!

В файле bot_logic ты найдешь все функции для бота

А сам бот создается в файле main.py

Обязательно установи YOLAv3 по ссылке https://github.com/ultralytics/yolov3/releases/download/v9.5.0/yolov3.pt. После чего закинь его в папку с проектом.

🚀 Возможности

- Распознавание людей на изображении с помощью YOLOv3.
- Подсчёт количества людей.
- Предупреждение при превышении порога (по умолчанию 20 человек).
- Работа через Telegram: просто отправьте фото боту.
- Автоматическое удаление временных файлов после обработки.
